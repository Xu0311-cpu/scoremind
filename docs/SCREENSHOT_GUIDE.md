## Screenshot Checklist

- Use a valid `.musicxml` sample.
- Confirm backend and frontend are running locally.
- Capture both Student Analysis and Technical Evidence.
- Include at least one screenshot showing the score preview and one showing the Learning Report.
- Do not imply PDF/image/OMR or real LLM support in captions.
